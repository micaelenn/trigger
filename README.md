# :zap: Trigger Extension

A local extension to save and trigger webhooks

## Requirements
- [Node.js](https://nodejs.org/en) - `18.0.0 +`

## Getting Started

Install dependencies

```
npm install
```

Development mode

```
npm run dev
```

Production build

```
npm run build
```

## Technologies
- [Vite](https://vitejs.dev/)
- [Styled Components](https://styled-components.com/)
